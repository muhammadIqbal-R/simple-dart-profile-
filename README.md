# Tugas Profil Singkat

Aplikasi halaman profil sederhana menggunakan Flutter dan Dart. 
Tugas ini dibuat untuk memenuhi pembelajaran dasar pengembangan aplikasi mobile 
dengan menerapkan widget fundamental, layouting, dan interaksi pengguna.

---

## Fitur

- **Tampilan Profil**: Menampilkan foto, nama, NIM, dan program studi
- **Edit Profil**: Tombol aksi yang memunculkan bottom sheet formulir edit
- **Hubungi Saya**: Dialog interaktif dengan pilihan kontak (Email, WhatsApp, Telepon)
- **Notifikasi**: SnackBar sebagai umpan balik aksi pengguna

---

## Teknologi

- Flutter SDK
- Dart
- Material Design 3

---

## Struktur Folder

```
tugas_profil_singkat/

├── assets/

```

---

## Cara Menjalankan

1. Pastikan Flutter SDK sudah terinstall
2. Clone atau buka project ini
3. Jalankan perintah:
   ```bash
   flutter pub get
   flutter run
   ```
